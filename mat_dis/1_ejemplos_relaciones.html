Ejemplos Relaciones


Determine si son o no son Simetrica, Antisimetrica y/o Reflexiva.


Problema 1


X = { 1,2,3,4 }

R = { (1,2), (2,2), (3,4), (4,1) }

  • No es Reflexiva
  • No es Simetrica
  • Si es Antisimetrica


Problema 2


T = { x,y,z }

R = { (x,x), (y,y), (z,z) }

  • Si es Reflexiva
  • Si es Simetrica
  • Si es Antisimetrica
