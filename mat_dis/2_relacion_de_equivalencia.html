Relacion de Equivalencia

Introduccion

Una relacion R en un conjunto A se llama relacion de equivalencia si es reflexiva, simetrica y transitiva (las tres).

Una particion (P) de un conjunto no vacio A es una coleccion de subconjuntos no vacios de A tales que:

  1. Cada elemento de A pertenece a uno de los conjuntos de P
  2. Si los subconjuntos A1 y A2 tienen elementos distintos de P, entonces A1nA2 = nulo (vacio.)

Los conjuntos que hay en P se llaman bloques o celdas de la particion. La siguiente figura muestra una particion de A enn 7 bloques

P = { A1, A2, A3, A4, A5, A6, A7 }



Ejemplo 1


Sea X = {a,b,c,d,e,f,g,h}

Considera los siguientes subconjuntos de X para determinar las particiones que surjan.

  • X1 = {a,b,c,d}
  • X2 = {a,c,e,f,g,h}
  • X3 = {a,c,e,g}
  • X4 = {b,d}
  • X5 = {f,h}

Respuesta

  • P1 = { x2, x4 }
  • P1 = { x3, x4, x5 }




Si P es una particion de un conjunto A, entonces P puede usarse para construir una relacion de equivalencia en A como sigue:

aRb si y solo si 'a' y 'b' son miembros del mismo bloque

'R' es una relacion de equivalencia en A porque:

  1. Si aEA, entonces es claro que 'a' esta en el mismo bloque, por lo tanto aRa
  2. Si aRb, entonces 'a' y 'b' estan en el mismo bloque, por lo que bRa
  3. Si aRb y bRc, enconces, 'a', 'b' y 'c' deben estar todos en el mismo bloque de P, en consecuencia aRc

Puesto que la relacion anterior es reflexiva, simetrica y transitiva, R es una relacion de equivalencia; a esta relacion se le denomina 'relacion de equivalencia determinada por la particion'



Ejemplo 2


Sea A = {1,2,3,4}, considera P = { {1,2,3},{4} }

Obten la relacion de equivalencia determinada por la particion P

Respuesta

AxA = { (1,1), (1,2), (1,3), (1,4), (2,1), (2,2), (2,3), (2,4), (3,1), (3,2), (3,3), (3,4), (4,1), (4,2), (4,3), (4,4) }

R = { (1,1), (1,2), (1,3), (2,1), (2,2), (2,3), (3,1), (3,2), (3,3), (4,4) }
