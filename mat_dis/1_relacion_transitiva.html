Relacion Transitiva

Una relación binaria R sobre un conjunto A es transitiva cuando se cumple:

“siempre que un elemento se relaciona con otro y éste último con un tercero, entonces el primero se relaciona con el tercero.”

Esto es:

  • Si aRb ˆ bRc ..entonces.. aRc

Dado el conjunto A y una relación R, esta relación es transitiva si: aRb y bRc se cumple aRc.

Ejemplos


  1. A = { 1,2,3,4 }

    R1 = { (1,1), (1,2), (1,3), (1,4), (2,2), (2,3), (2,4), (3,3), (3,4), (4,4) }

    • Si es Transitiva

  2. A = { a,b,c,d }

    R1 = { (a,a), (b,c), (c,b), (d,d) }

    • No es Transitiva

  3. X = { 1,2,3,4 }

    R1 = { (1,2), (2,2), (3,4), (4,1) }

    • No es Transitiva

  4. X = { 1,2,3,4 }

    R1 = { (1,2), (1,3), (4,2) }

    • Si es Transitiva
