Matrices de Relaciones

Introduccion

Una matriz es una manera conveniente de representar una relacion ‘R’ de ‘A’ a ‘B’. Se etiquetan las columnas con elementos de ‘B’ en algun orden arbitriario.

Despues, el elemento en el renglon ‘x’ y la columna ‘y’ se hace igual a uno si xRy (x se relaciona con y); y a cero, de no tener relacion. Esta matriz se llama matriz de relacion R.



Ejemplo 1:


A = { 1,2,3,4 } , B = { a,b,c,d }

R1 = { (1,b), (1,d), (2,c), (3,c), (3.b), (4,a) }

Elabore

  1. Diagrama de flechas
  2. Matriz para la relacion R1.
  3. Matriz relativa a los ordenes: ‘2,3,4,1’ y ‘d,b,a,c’
