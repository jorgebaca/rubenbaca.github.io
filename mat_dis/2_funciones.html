Funciones (Unidad II)

Coneptos Basicos

Si biajamos durante cierto tiempo a una velocidad constante, sabemos que d=(v)(t) . Entonces si viajamos a 55 km/hr durante 't' horas, d=55(t) , donde t=tiempo, d=distancia

La ecuacion anterior define una funcion. Una funcion asigna a cada miembro de un conjunto X exactamente un miembro de un conjunto Y. (X y Y pueden ser el mismo conjunto).

La funcion definida asigna a cada numero no negativo 't' el valor 55(t).

Por ejemplo, el tumero t=1 se asigna al valor '55'... el numero t=3.45 se asigna al valor '189.75'.

Estas asignaciones se pueden representar como pares ordenados:

(1, 55) , (3.45, 189.75)

Dentro del conjunto de todas las relaciones que se pueden definir de un conjunto A a un conjunto B, son importantes aquellas en las que cada elemento de A esta relacionado con un unico elemento de B. A estas relaciones se les llama funciones.


Sean A y B dos conjuntos no vacios, una funcion 'f' de A a B es un subconjunto del prodcto cartesiano AxB que tiene la propiedad de que para cada a∑A, existe exactamente una b∑B con (a,b)∑f. Cada elemento de A aparece exactamente una vez como primer componente de un par ordenado de la relacion.

En ocasiones denotamos una funcion f:A→B (f de A a B).



Ejemplo


Determinar si 'f' es funcion o no, con f = { (1,a), (2,b), (3,a) }

de X = { 1,2,3 } a Y = { a,b,c }

Respuesta

  • Si es funcion


Una funcion puede representarse con un diagrama de flechas, para la cual se refiere que haya una flecha justo desde cada elemento del dominio.

La definicion de funcion permite volver a usar los elementos de B, ademas la definicion no requiere que todos los elementos de B se utilizen.



Ejercicio


Determinar si son o no son funciones

  • El conjunto { (1,a), (2,b), (3,c), (1,b) } con X = { 1,2,3,4 } y Y = { a,b,c }

    • Respuesta: No es funcion

  • Sean X = { 1,2,3 } y Y = { a,b,c }, R = { (1,a), (2,b), (3,c), (1.b) }

    • Respuesta: No es funcion
