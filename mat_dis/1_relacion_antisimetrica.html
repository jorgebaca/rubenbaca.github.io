Relacion Antrisimetrica

Una relacion en un conjunto A se llama antisimetrica si para toda a,b∑, si (a,b)∑R y a≠b, entonces (b,a)!∑R; es decir:

Si en todos los casos en que a∑b y b∑a se tiene que a=b. Esto es si a≠b, entonces (a,b)!∑R, o (b,a)!∑R.

La digrafica de una relacion antisimetrica tiene la propiedad de que entre cualesquiera dos vertices existe a lo sumo una arista dirigida.



Ejemplo


Determine si es o no es antisimetrica.

R1 = { (1,1), (1,2), (1,3), (1,4), (2,2), (2,3), (2,4), (3,3), (3,4), (4,4) }

  • Si es Antisimetrica

R2 = { (a,a), (b,c), (c,b), (d,d) }

  • No es antisimetrica
