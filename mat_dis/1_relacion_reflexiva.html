Relacion Reflexiva

Una relacion de un conjunto A es reflexiva si (a,a)∑R para todos los valores de a∑A. Es decir, si aRa para todas a∑A.

Esto es, R es reflexiva si cada uno de los elementos de A esta relacionado con sigo mismo.

Podemos reconocer facilmente una relacion reflexiva si:

  • La digrafica de una relacion reflexiva tiene un laso en cada vertice.
  • La matriz tiene “unos” en la diagonal principal.


Ejemplo 1


Determine qe la siguiente relacion es relexiva.

A = { 1,2,3,4 }

Obten la relacion definida por (a,b)∑R, si a≤B y a,b∑A.

Ademas traza la grafica dirigida y la matriz.

Respuesta:

R = { (1,1), (1,2), (1,3), (1,4), (2,2), (2,3), (2,4), (3,3), (3,4), (4,4) }
