Diagrama de Flechas

Introduccion

Existe otra manera de visualizar una relacion y es a traves de una representacion grafica, donde se destacan los puntos que pertenecen a ‘A’ y los puntos que pertenecen a ‘B’.

Se trazan flechas que indican la relacion que existe entre cada elemento del conjunto ‘A’ y su correspondiente en el conjunto ‘B’.

A esta representacion grafica se le conoce como diagrama de flechas.

Es importante tener en cuenta lo siguiente:

  • Conjunto inicial / de partida: Es del que salen las flechas.
  • Conjunto final / de llegada: Es al que llegan las flechas.
  • Conjunto Original: Es el formado por los elementos del conjunto inicial de los que parte alguna flecha
  • Conjunto Imagen: Es el formado por los elementos del conjunto final a los que llega alguna flecha. Conj. Inicial


Ejemplo


A = { a,b,c } , B = { 1,2,3,4 }

R = { (a,2), (b,2), (b,3), (c,4) }

Obtener diagrama de flechas, conjunto inicial, final, original e imagen.

Respuesta

  • Conj. Inicial = { a,b,c }
  • Conj. Final = { 1,2,3,4 }
  • Conj. Original = { a,b,c }
  • Conj. Imagen = { 2,3,4 }
