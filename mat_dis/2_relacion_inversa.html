Relacion Inversa

Introduccion

Un tipo diferente de operacion en una relacion R de A a B es la formacion de la inversa (R-1)

La relacion R-1 es una relacion de B a A que se define por:

bR-1a si y solo si aRb



Ejercicio


A = {1,2,3,4} , B = {a,b,c}

R = { (1,b), (2,b), (2,c), (3,b), (4,b) }

S = { (1,b), (2,c), (3,b), (4,b) }

Determinar:

  • R-1
  • S-1

Respuesta:

  • R-1 = { (b,1), (b,2), (c,2), (b,3), (a,4), (a,1) }
  • S-1 = { (b,1), (c,2), (b,3), (b,4) }
