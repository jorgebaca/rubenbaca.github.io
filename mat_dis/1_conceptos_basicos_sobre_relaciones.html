Matematicas Discretas

Surge como una disciplina que unifica diversas areas tradicionales de las matematicas (combinatoria, probabilidad, aritmetica, grafos, etc) como consecuencia de su interes en la informatica y las telecomunicaciones. La información se manipula y almacena en los ordenadores en forma discreta (palabras formadas por ceros y unos). Se necesita contar objetos, se precisa estudiar relaciones entre conjuntos finitos (busquedas de bases de datos), es necesario analizar procesos que incluyan un numero finito de pasos.

La matematica discreta proporciona algunas bases matematicas para otros aspectos: estructuras de datos, teoria de automatas, sistemas operativos, investigacion operativa, etc; asi como ayuda al desarrollo de ciertas capacidades para un ingeniero, como: capacidad de formalizar, de razonar, etc.

Para hacernos una idea mas clara del contenido de esta disciplina, podemos plantearnos algunas preguntas en informatica y responderlas con metodos de matematica discreta, por ejemplo:

  • Dada una tecnologia de cableado, Cual es el diseño de red mas economico?
  • Cuantas palabras clave validas hay para acceder a un sistema?
  • Como modificar adecuandamente un mensaje?

La matematica discreta nacio hace pocos años , como concecuendia de las computadoras (maquinas finitas).

Relaciones

Nuestro mundo esta lleno de relaciones:

  • Entre personas
  • Cantidades
  • Etc

Al definir una relacion, estamos estableciendo un vinculo entre elementos de un conjunto y otro, o bien; del mismo conjunto.

Podemos pensar en una relacion de un conjunto a otro como una tabla que lista los elementos del 1er conjunto relacionados con los elementos del 2o conjunto.

Relacion de estudiantes-cursos



Ejemplo


Sean A = { 1,2,3 } y B = { r,s }

AxB = { (1,r), (1,s), (2,r), (2,s), (3,r), (3,s) }

BxA = { (r,1), (r,2), (r,3), (s,1), (s,2), (s,3) }

Algunas relaciones podrian ser:

R1 = { (1,r), (1,s), (2,r), (2,s), (3,r), (3,s) }

R2 = { (2,r), (2,s), (3,r) }



Dominio y Rango


El conjunto:

{ a∑A / (a,b) ∑ para alguna b∑B }

se llama dominio de R: Dom(R). Esto es, el dominio corresponde al conjunto de elementos de A que estan relacionados con algun elemento de B.

El conjunto:

{ b∑B / (a,b) ∑ para alguna a∑A }

se llama rango de R: Ran(R). Es decir, el rango es el conjunto de todos los elementos de B relacionados con algun elemento de A.

Ejemplo de Dominio y Rango

R1 = { (1,r), (1,s), (2,r), (2,s), (3,r), (3,s) }

Dom(R1) = { 3,2,1 }

Ran(R1) = { r.s }

Nota: Es posible establecer una relacion con solo especificar que pares ordenados pertenecen a la misma, Sin embargo, algunas veces es pocible definir una relacion dando una regla para partenecer a la relacion.
