Funcion de Dispercion

Introduccion

Las funciones que implican el operador modulo desempenan un papel importante en matematicas y computacion. La definicion es:

Operador Modulo:
Si 'x' es un entero positivo, y 'y' es un entero positivo; se define x mod y como el residuo cuando 'x' se divide entre 'y'.

  • 6 mod 2 = 0
  • 5 mod 1 = 0
  • 8 mod 12 = 8
  • 158 mod 7 = 4

Ahora bien, imagina que se tienen celdas en la memoria de una computadora, (indexadas del 0 al 10), como se muestra a continuacion:

Se desea guardar y recuperar enteros no negativos arbitrarios en estas celdas. Uno de los enfoques para esto es usar una funcion de dispercion. Una funcion de dispercion toma un dato que debe guardarse o recuperarse, calcula la primera opcion para la ubicacion del dato empleando el operador modulo.

Por ejemplo, para guardar o recuperar el numero 'n' se puede tomar para la localizacion 'n mod 11'. La funcion de dispercion se convierte en h(n) = n mod 11
