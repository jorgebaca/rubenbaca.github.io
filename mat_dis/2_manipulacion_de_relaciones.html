Manipulacion de Relaciones

Igual que puede manipularse a trabajarse con los numeros y formulas aplicando las reglas del algebra, tambien pueden definirse operaciones que permitan manipular a las relaciones

Con estas operaciones se pueden cambiar, combinar y refinar relaciones existentes para producir otras nuevas.

Sean R y S las relaciones de un conjunto A a un conjtno B (recordando que R y S son simplemente subconjuntos de AxB), es posible usar operaciones de conjuntos en R y S.

Por ejemplo, el complemento de R (R') se menciona como la relacion complementaria. Por supuesto, es una relacion de A o B que puede expresarse como sige:

aR'b si y solo si a!Rb

Tambien puede formarse la interseccion RnS y la union RuS de las relaciones R y S. Todas las operaciones de conjuntos pueden ser utilizados de esta manera para producir nuevas relaciones.

Ejemplo

A = {1,2,3,4} , B = {a,b,c}

R = { (1,b), (2,b), (2,c), (3,b), (4,b) }

S = { (1,b), (2,c), (3,b), (4,b) }

Calcular

  1. R'
  2. RnS
  3. RuS

Respuesta

Producto Cartesiano (opcional)

AxB = { (1,a), (1,b), (1,c), (2,a), (2,b), (2,c), (3,a), (3,b), (3,c), (4,a), (4,b), (4,c) }

  1. R' = { (1,c), (2,a), (3,a), (3,c), (4,b), (4,c) }
  2. RnS = { (1,b), (2,c), (3,b) }
  3. RuS = { (1,b), (1,c), (2,c), (3,b), (4,b), (1,a) }
